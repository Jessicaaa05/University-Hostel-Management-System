# 🏠 University Hostel Management System

A database management system developed using **Microsoft Access** to streamline university hostel administration, including student accommodation, room allocation, visitor registration, payments, maintenance services, transportation scheduling, and security monitoring.

## 📖 Overview

The University Hostel Management System was developed as part of the **DIT5201 Database Systems** course at Multimedia University (MMU).

The system aims to replace traditional paper-based hostel administration with a centralized database solution. By digitizing records and management processes, the system improves efficiency, data consistency, accessibility, and overall administrative productivity.

---

## 🎯 Project Objectives

* Develop an integrated database system for hostel management.
* Improve operational efficiency compared to manual record keeping.
* Apply database design concepts such as ERD, relational modelling, forms, reports, and normalization.
* Gain practical experience in database development using Microsoft Access.

---

## ❗ Problem Statement

Manual hostel management processes often lead to:

* Data redundancy and duplication
* Slow information retrieval
* Human errors in record management
* Difficult maintenance and updates
* Limited data security

This project addresses these challenges by providing a centralized computerized database system for managing hostel operations.
